입금 전 관리

입금 전 관리 메뉴

고객이 무통장 입금으로 결제한 주문 확인하는 메뉴, 입금 확인 또는 주문 취소 가능

입금 확인하기

입금 전 관리 메뉴에서 주문 선택 후 입금 확인 > 배송준비중 (또는 상품준비중) 으로 주문 상태 변경

입금 전 취소하기

입금 전 관리 메뉴에서 주문 선택 후 주문 취소 > 주문 상태가 입금전 취소로 변경되며

취소된 주문은 [관리자] 주문 > 취소/교환/반품/환불 > 입금 전 취소 관리에서 확인 가능

미입금 주문 자동 취소 설정하기

[관리자] 쇼핑몰 설정 > 주문 설정 > '주문 후 설정' > 미입금 주문 자동 취소 사용함 선택 후 기한 설정 > 페이지 하단 저장

배송중 처리하기

배송준비중 관리 메뉴에서 주문 선택 > 운송장 정보의 택배사 선택/송장번호 기재 후 체크박스 체크 > 배송중처리 버튼 클릭

배송대기 처리하기

배송준비중 관리 메뉴에서 주문 선택 > 운송장 정보의 택배사 선택/송장번호 기재 후 송장 번호 저장 > 주문이 배송 대기 관리로 이동

묶음 배송 처리하기

수령자 정보가 동일한 주문건 일괄 선택 > 체크박스 체크하여 묶음 배송 처리 버튼 클릭

> 묶음배송 탭에서 운송장 정보의 택배사 선택/송장번호 기재하여 배송중처리 버튼 클릭

엑셀 배송 처리하기

배송준비중 관리 메뉴 하단 엑셀 일괄 배송 처리에서 엑셀 양식 다운로드 > 엑셀 수정 후 쇼핑몰 선택/배송사 선택/파일선택 > 등록 (가이드)

배송중 처리하기

배송 대기 관리 메뉴에서 주문건 체크박스 선택 > 배송중 처리 버튼 클릭 > 주문이 배송중 관리 메뉴로 이동

※ 연동택배 주문건

배송대기는 택배 연동 서비스 사용시 주로 사용하며, 연동택배 주문건의 경우 임의로 주문 상태를 변경하지 않고

배송대기까지 처리하면 1~2시간 내 연동된 택배 시스템에 전달 > 이후 실제 택배 배송 상태에 따라 자동으로 상태 변경됨

배송준비중 되돌리기

배송 대기 관리 메뉴에서 주문 선택 후 배송준비중 처리 버튼 클릭 > 주문이 배송준비중 관리 메뉴로 이동

배송완료 조회하기

배송 완료 조회 메뉴에서 원하는 조건 선택하여 검색 시 배송완료된 주문 조회 가능

※ 배송완료 메뉴의 경우 이전 상태값(배송중)으로 되돌리기 불가, 필요 시 사유/주문번호 기재하여 1:1 문의 접수 필요

배송완료 자동체크 기능 (유료서비스)

택배 연동을 사용하지 않으나 배송중 주문을 자동으로 배송완료 처리 하고 싶은 경우 배송완료 자동체크 앱 사용 가능

자동 처리 가능한 9개의 택배사 주문건이라면 택배 배송상황에 맞춰 자동으로 배송완료 처리됨

앱 > 앱스토어 바로가기 > 배송완료 자동체크 검색하여 이동하기 > 동의함 > 설치완료

설치 후 처리 건수 충전 및 자동체크 활성화 하여 이용 가능

> 기본 또는 상세신청 항목 표시 > 취소/교환/반품 신청 버튼 노출 범위 설정
> 취소신청 항목 중 배송전/배송후 원하는 배송상태 체크하여 설정
가이드 : 취소 관리

취소관리

메뉴 경로 : [관리자] 주문 > 취소/교환/반품/환불 > 취소 관리

취소접수하기

- 구매자 취소 신청 주문 : 취소 관리 메뉴에서 취소할 주문건 취소처리의 처리 버튼 클릭 > 취소처리 팝업에서 상품 선택 후 취소 접수 > 구분 선택 후 확인
- 운영자 취소 처리 주문 : 전체주문조회 메뉴에서 주문번호 클릭 > 주문상세정보 창 주문 취소 > 취소처리 팝업에서 상품 선택 후 취소 접수 > 구분 선택 후 확인

취소완료하기

취소 관리 메뉴에서 주문상태 항목 취소처리 중인 경우 환불처리까지 진행해야 취소 완료 상태로 변경됨

취소 관리 메뉴에서 주문번호 클릭 > 환불 탭에서 환불처리/상세의 처리 버튼 클릭 > 환불완료처리

※ 무통장 주문건의 경우 자동환불이 되지 않기 때문에 운영자가 구매자에게 직접 송금 필요
※ 카드결제 주문건의 경우 환불완료 처리 및 신용카드 취소 처리 필요, 결제수단별 환불방법은 환불관리 메뉴 내용 참고

취소철회하기

취소관리 메뉴에서 취소처리의 상세 클릭 > 취소처리 상세정보창에서 취소철회 클릭 > 구분 선택, 사유 기재 후 철회완료

교환 관리

배송전 교환완료하기

- 금액 변동 없는 경우

접수되면 자동 교환 완료 상태가 되고, 교환된 상품을 포함해 배송처리 필요

- 차액 추가 결제 필요한 경우

더 비싼 상품으로 교환했다면 고객에게 차액을 무통장 입금으로 결제 하도록 안내

> 입금 확인 후 입금전 관리 메뉴에서 해당 주문 선택하여 입금 확인 > 이후 일반 상품과 동일하게 배송처리
- 차액 환불이 필요한 경우

더 저렴한 상품으로 교환했다면 고객에게 차액을 환불해 주고, 이후 일반 상품과 동일하게 배송처리

교환관리

배송후 교환완료하기

- 금액 변동이 없는 경우

고객에게 배송된 상품 반송 요청 > 상품 회수 시 [관리자] 주문 > 취소/교환/반품/환불 > 교환 관리 메뉴에서 주문건 체크 후 수거완료 버튼 클릭

> 이후 교환 상품 배송처리
- 차액 추가 결제 필요한 경우

더 비싼 상품으로 교환하거나 왕복 배송비가 추가되면 고객에게 차액 결제 안내 및 상품 반송 요청 > 입금 확인되면 [관리자] 주문 > 입금전 관리에서

해당 주문건 입금 확인 > 상품 회수되면 [관리자] 주문 > 취소/교환/반품/환불 > 교환 관리에서 주문건 선택 후 수거완료 버튼 클릭 > 이후 교환 상품 배송처리

- 차액 환불이 필요한 경우

저렴한 상품으로 교환했다면 고객에게 차액 환불 후 배송된 상품 반송 요청 > 상품 회수 후 [관리자] 주문 > 취소/교환/반품/환불 > 교환 관리에서

주문건 선택 후 수거완료 버튼 클릭 > 이후 교환 상품 배송 처리

교환관리

교환 철회하기

[관리자] 주문 > 취소/교환/반품/환불 > 교환 관리에서 철회하려는 주문건 교환처리 항목의 상세 버튼 클릭

> 교환처리 상세정보 창에서 교환철회 버튼 클릭 > 교환 철회 구분 선택 후 철회완료
※ 금액 변동이 있는 교환 철회하는 경우 유의해야 할 사항

교환처리 중 추가 입금이 발생되면 고객에게 다시 입금 받은 후 철회 필요

교환처리 중 추가 환불 발생 시 고객에게 직접 환불 후 철회 필요

교환처리 중 PG 취소 발생 시 PG사 관리자페이지에서 승인 취소 철회 가능한지 확인 후 철회 진행 필요

교환관리

고객 직접 반품신청 가능하도록 설정하기

[관리자] 쇼핑몰 설정 > 주문 설정 > '주문 정책 설정' > '주문 후 설정' > '구매자 취소/교환/반품 신청 설정'

> 취소/교환/반품 신청버튼 노출 범위 설정 > 반품신청 항목에 배송후 원하는 배송상태 체크하여 설정

반품 접수하기

- 구매자 반품 신청 주문건

반품 관리에서 반품신청된 주문건 반품 처리 항목의 처리 버튼 클릭 > 반품처리 팝업창에서 상품 선택하고 반품접수 > 반품 접수 구분 선택 후 확인 클릭

- 운영자 반품 처리 필요 주문건

전체주문조회 메뉴에서 주문번호 클릭 > 주문상세정보창에서 상품 반품 클릭 > 반품처리 팝업창에서 상품 선택 후 반품 접수 클릭 > 반품 접수 구분 선택 후 확인

1. '구매 확정' 이란?

구매 확정은 이후에 취소, 교환, 반품 의사가 없다는 뜻으로, 구매 확정한 상품은 취소, 교환, 반품 처리 불가

상품이 배송 중 또는 배송 완료 상태일 때, 구매자가 '구매 확정' 처리 가능

단, 불가피한 상황이라면 운영자가 구매 확정을 취소한 후 CS 처리를 진행 필요

※ 유의사항 : 외부 주문(오픈마켓, 네이버페이, 카카오페이) 은 어드민에서 구매 확정으로 바꿔도 연동되지 않음
2. 구매 확정 설정/사용 방법

1) 사용 설정하는 방법

[관리자] 쇼핑몰 설정 > 주문 설정 > 주문 정책 설정 > 주문 후 설정에서 '구매 확정 설정'

구매 확정 버튼을 '사용함'으로 선택하면, 쇼핑몰 '주문 조회' 목록에 구매 확정 버튼이 표시됨

2) 구매 확정 처리하는 방법(구매자)

상품이 배송 중, 배송 완료 상태면 쇼핑몰 마이 페이지의 '주문 조회' 목록에 구매 확정 버튼이 표시되며 버튼 클릭 시 구매확정 상태로 변경

구매확정 기능

3) 구매 확정 처리하는 방법(운영자)

3-1) 수동 처리

주문 번호 검색 후 주문 상세정보창에서 주문 상태 탭의 '상태 변경' 버튼을 눌러 운영자가 구매 확정 처리 가능

3-2) 자동 처리

'구매 확정 자동체크' 기능을 통해 운영자가 설정한 특정 기간이 지나면 자동으로 '구매확정' 상태가 됨

단, 배송 완료일이 최근 1개월 이내인 주문만 자동으로 구매확정 처리 가능

4) 구매 확정 취소하는 방법

운영자가 주문 상세 정보창의 주문 상태 탭에서 '구매 확정 취소' 버튼 클릭하여 취소 가능

구매 확정한 주문을 취소, 교환, 반품 처리해야 할 때 사용 가능

5) 구매 확정 검색하는 방법

구매 확정된 주문은 '전체 주문 조회'와 '배송 완료 조회' 메뉴에서 주문 상태를 '구매 확정'으로 하여 검색 가능

네이버페이 주문의 배송처리

- 네이버페이 주문 배송중 처리

[관리자] 주문>배송준비중 관리에서 네이버페이 주문 선택 후 송장번호와 택배사 선택하고 배송중 처리 (일반 주문과 동일하게 처리)

단, 네이버페이 주문관리는 네이버페이센터에서 진행하는 것을 권장함

또한 일정기간 동안 배송지연처리 없이 배송진행되지 않는 경우 배송지연으로 네이버페이측에서 패널티를 부과할 수 있음

재고 부족 등으로 배송 지연되는 경우 네이버페이 센터에서 배송보류처리 진행 필요

- 네이버페이 주문 취소 처리

전체주문조회 등 메뉴에서 취소하려는 네이버페이 주문번호 클릭 > 주문상세정보창에서 주문취소 클릭

> 주문취소 처리창에서 취소접수 > 취소 사유 입력하여 취소처리 완료
- 네이버페이 주문 교환 처리

네이버페이 주문은 네이버페이센터에서만 교환처리 가능. 또한 네이버페이는 교환시 동일 상품만 교환 가능하며 옵션 변경 불가

네이버페이센터에서 수거완료 처리 시 카페24 기존 주문은 교환완료 상태가 되며 교환처리로 생성된 주문은

재배송준비중 상태로 수집됨. 교환처리된 주문은 구매자가 구매확정 시 카페24 주문 상태도 배송완료 상태로 변경됨

- 네이버페이 주문 반품 처리

구매자가 반품 요청한 경우에만 카페24에서 반품 처리 가능

카페24에서 반품처리 시 네이버페이센터에서 승인하지 않아도 상품 반품 및 환불까지 한번에 진행되기 때문에 반드시 상품 수거 확인 후 승인 처리 필요

네이버페이 주문건의 경우 전체 품목 반품접수되더라도 특정 품목만 선택하여 반품처리 가능하나

반품철회 및 반품거부처리는 네이버페이 센터에서만 가능하며 카페24에서는 불가함

※ 네이버페이센터에서 반품처리 필요한 경우 : 반품 배송비 차감 등 부분환불 필요한 경우, 반품 철회 필요한 경우 , 구매확정이 된 주문건

- 쿠키
