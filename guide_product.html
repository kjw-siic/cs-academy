상품 등록

일반 등록
개별 상품 직접 입력 · 등록 개수 제한
▼
[관리자] 상품 > 상품 등록 > 일반등록
1등록 방법
  • 일반등록 메뉴에서 항목별 설정 입력 → 하단 상품 등록 버튼 클릭
  • 항목별 입력 방법은 관련 가이드 참고
2상품/분류 등록 개수 제한
단계상품 개수분류 개수
정상8만개 미만2천개 미만
주의8만~10만개 미만2천~3천개 미만
경고10만개 이상3천개 이상
⚠️
등록 개수가 주의/경고 단계에 도달하면 쇼핑몰 속도 저하 및 운영 이슈 발생 가능성이 높아짐.
📊
엑셀 등록/수정
대량 상품 일괄 등록 · 수정 · 옵션/재고 수정
▼
[관리자] 상품 > 상품 관리 > 상품 엑셀 관리
1상품 등록
등록용 엑셀 다운로드 → 엑셀 수정 → 엑셀 업로드
2상품 수정
  • 상품목록 메뉴에서 기본양식 선택 → 엑셀 다운로드
  • 엑셀 수정 후 상품 엑셀 관리 메뉴에서 업로드
3옵션/재고 수정
  • 상품목록 → 옵션/재고 초기화 양식 선택 → 엑셀 다운로드
  • 옵션정보 수정 후 상품 엑셀 관리 메뉴의 옵션 수정 또는 재고 수정 탭에서 업로드

에디봇
AI 기반 상품 상세설명 에디터
▼
[관리자] 상품 > 상품 등록 > 일반등록
1에디봇이란?
  • 상품 상세설명 콘텐츠를 빠르게 완성하는 AI 콘텐츠 에디터 툴
2주요 기능
  • AI 기반 이미지 분석 및 자동 분류
  • 블로그 작성하듯 친숙한 인터페이스
  • 프레임 · 스티커로 상품 이미지 꾸미기
  • 에디봇핏 — 사이즈·소재·세탁법 등 상품정보 템플릿화
  • 영어·일본어·중국어 등 자동 번역 지원
  • 완성도 확인용 미리보기 기능

옵션 설정
옵션 등록 · 연동 옵션 · 조합분리선택형
▼
[관리자] 상품 > 상품 옵션 관리 > 품목 생성형 옵션 (상품 연동형 옵션)
1옵션 관리 메뉴에서 옵션 생성
  • 품목생성형 옵션 또는 상품연동형 옵션 → 옵션 등록 클릭
  • 옵션 정보 입력 후 등록 → 옵션 생성 완료
2상품 등록/수정창에서 옵션 등록
[상품] 상품등록 > 일반등록 > 옵션/재고 설정
  • 옵션사용 사용함 → 옵션 구성방식 선택 → 직접 입력하기
  • 옵션명/옵션값 기재 → 모든 옵션 품목 추가
3첫번째 옵션에 따라 두번째 옵션 다르게 설정
  • 옵션/재고설정 → 옵션사용 사용함
  • 옵션 구성방식 조합분리선택형 → 직접 입력하기 → 모든 옵션 품목 추가
  • 하단 추가된 품목 중 불필요한 조합 선택 후 삭제
  • 상품 등록/수정 완료
💡
예시: 색상(빨강/파랑) × 사이즈(S/M/L)에서 "빨강-S" 조합만 빼고 싶을 때 사용.

상품 재고 관리
재고 설정 · 수량체크 기준 · 품절 표시
▼
[관리자] 상품 > 상품관리 > 재고관리 > 상품재고 관리
1재고 설정 방법
상품 등록/수정창 > 옵션/재고설정 항목
  • 재고관리 사용: 사용함
  • 수량 체크 기준: 결제기준 또는 주문기준
  • 재고수량 입력 + 품절 표시 체크
2수량 체크 기준 선택
기준설명
결제기준실제 결제 완료된 주문건에 대해서만 재고 차감
주문기준주문 접수 시점에 재고 차감 (결제 미완료도 포함)
⚠️
결제 미완료 주문까지 재고 차감을 원치 않으면 결제기준, 재고가 확정적으로 빠지길 원하면 주문기준으로 설정.
🚫
재고관리 사용만 체크하고 품절표시 사용에 체크하지 않으면, 재고가 0이 되어도 자동 품절 처리되지 않음.

분류 관리
대분류·중분류·소분류 생성 · 수정 · 이동
▼
[관리자] 상품 > 상품 관리 > 상품 분류 관리
1카테고리 생성
  • 대분류 추가 버튼으로 대분류 생성
  • 추가된 분류명 선택 → 우측 + 버튼으로 중분류 생성 (동일 방법으로 소분류)
2분류명 수정
  • 대분류 또는 중분류 선택 → 오른쪽 분류명 항목에서 수정
3분류 삭제 / 이동
  • 삭제: 분류 선택 → x 삭제 버튼
  • 이동: 분류 선택 → 상/하 드래그 앤 드롭 → 분류이동저장
🏪
분류별 진열
카테고리별 상품 진열 · 순서 조정
▼
[관리자] 상품 > 상품 진열 > 분류별 진열
1카테고리에 상품 진열
  • 상품목록 → 진열할 상품 선택 → 분류수정
  • 팝업에서 분류 선택 [+추가] → 선택한 분류로 변경 → 확인
2분류 진열 방법 수정
  • 상품 분류 관리 → 수정할 분류 선택
  • 상품진열 항목 → 사용자지정 또는 자동정렬+사용자지정 선택 → 확인
3분류 내 상품 순서 수정
  • 분류별 진열 메뉴에서 분류 선택 → 검색
  • 순서 변경할 상품 선택 → 진열순위에 숫자 기재 → 확인
🔄
메인 진열 · 자동 진열
메인분류 관리 · 자동 정렬 조건 설정
▼
[관리자] 상품 > 상품 진열 > 메인진열 / 자동 진열
1메인분류 생성
  • 메인진열 메뉴 → 메인분류 관리 버튼
  • 추가 또는 기존 분류명 수정 가능
2메인분류에 상품 진열
상품목록에서 진열
  • 상품 선택 → 메인진열 수정 → 팝업에서 메인진열 진열함 선택 → 확인
메인진열 메뉴에서 진열
  • 원하는 메인분류 선택 → 검색 → 오른쪽 상품추가 → 상품 체크 후 선택
3자동진열 설정
  • 메인진열 → 메인분류 관리 → 메인분류 선택 → 자동 진열 기능 사용함 → 저장
  • 자동 진열 메뉴 → 메인분류 선택 → 왼쪽 상세 조건 → 상품 데이터 화살표 클릭 → 정렬기준 선택
⏰
진열 예약
특정 기간 동안 상품 자동 진열/미진열
▼
[관리자] 상품 > 상품 진열 > 진열 예약
1진열 예약 설정
  • 진열 예약 메뉴 → 오른쪽 예약등록
  • 기간/상품 선택 → 진열할 카테고리 선택 → 저장
2미진열 상품 기간 한정 진열
  • 진열시작: 상품 상태 진열함으로 설정
  • 진열종료: 상품 상태 진열안함으로 설정 체크
📌
제한사항
• 한 예약건당 상품 분류 1개, 메인 분류 5개까지 설정 가능
• 동시에 다른 분류 예약 필요 시 예약 추가 등록 필요

삭제 상품 목록
상품 삭제 · 복구 · 완전 삭제 · 엑셀 다운로드
▼
[관리자] 상품 > 상품관리 > 삭제 상품 목록
1상품 삭제
  • 상품목록에서 불필요한 상품 선택 → x 삭제
  • 효과적인 관리 및 쇼핑몰 운영을 위해 불필요 상품은 완전 삭제 권장

  • 삭제 상품 목록에서 제거되며 이후 확인 불가

상품 등록/이미지 관련
이미지 용량 · 화질 · 수정 오류
▼
상품 수정 시 "Request Entity Too Large" 문구가 노출돼요
상품 수정 시 업로드하려는 이미지 또는 데이터 용량이 서버 허용 용량을 초과한 경우 발생. 일반적으로는 이미지를 인터넷에서 복사해온 경우 발생하므로, 이미지를 PC에 다운로드 받은 다음 직접 등록하시도록 안내 필요.
상품이미지가 원본과 다르게 흐릿하게 노출돼요
Cafe24는 상품 이미지 업로드 시 자동 리사이징이 적용되어 표시 사이즈보다 작은 이미지를 올리면 강제 확대되어 흐릿해질 수 있음. 원본 이미지를 권장 사이즈 이상으로 준비하여 업로드 필요.

옵션 관련
옵션 구성방식 차이 · 엑셀 등록 · 일괄 설정
▼
조합일체선택형, 조합분리선택형, 상품연동형, 독립선택형의 차이는?
유형설명
조합일체선택형모든 옵션을 하나의 드롭다운에서 조합된 형태로 선택 (예: 빨강/S)
조합분리선택형옵션별 드롭다운 분리, 첫 옵션에 따라 두 번째 옵션값 다르게 설정 가능
상품연동형여러 상품에 공통으로 쓰는 옵션을 미리 등록 후 연동
독립선택형옵션별 재고를 관리하지 않고 단순 선택만 받는 방식
상품 엑셀 등록을 통한 옵션 등록 및 수정(옵션 정보, 추가가격)은 어떻게 하나요?
상품목록에서 옵션/재고 초기화 양식 엑셀 다운로드 → 옵션명·옵션값·추가가격 수정 → 상품 엑셀 관리의 옵션 수정 탭에서 업로드. 상세 양식 규칙은 관련 가이드 참고 필요.
옵션 추가 가격 또는 재고 수량을 한번에 등록하고 싶어요
상품 엑셀 관리의 옵션 수정 또는 재고 수정 탭 활용. 엑셀 양식에 품목별 추가가격·재고수량 기재 후 업로드하면 일괄 반영됨.
📦
재고/판매 관련
실시간 품절 · 특정 고객 구매 제한
▼
재고에 따라 실시간으로 품절 처리하는 방법은?
상품 등록/수정창의 옵션/재고 설정에서 ①재고관리 사용 사용함, ②수량체크 기준 주문기준, ③품절표시 체크 세 가지 모두 설정 필요. 셋 중 하나라도 빠지면 자동 품절 처리되지 않음.
특정 고객만 구매 가능하도록 설정할 수 있나요?

상품 미노출 · 순서 설정 · 시간 예약
▼
쇼핑몰에 상품이 진열되지 않아요
체크 포인트:
• 상품 상태가 진열함으로 설정되어 있는지
• 판매함 상태인지
• 해당 분류에 등록되어 있는지
• 품절 또는 진열 기간 설정 여부
• 상품 노출 기간 내에 있는지
상품 진열 순서를 설정했는데 설정한 순서로 보이지 않아요
분류의 상품진열 방식 확인 필요. 자동정렬만 선택되어 있으면 사용자 지정 순서가 반영되지 않음. 사용자지정 또는 자동정렬+사용자지정으로 변경 필요.
원하는 시간에 상품을 진열할 수 있나요?
[상품] 상품 진열 > 진열 예약 기능으로 특정 기간에만 상품을 진열 가능. 미진열 상품을 해당 기간에만 진열하려면 진열시작에 진열함, 진열종료에 진열안함으로 설정.

▼
상품 정보를 일괄로 변경하려면 어떻게 해야 하나요?
[상품] 상품 관리 > 상품 정보 일괄 변경 메뉴에서 상품 선택 후 구분 박스에서 변경 항목 선택 → 일괄 변경. 판매가·배송정보·아이콘·소비자가 취소선 등 다양한 항목 일괄 수정 가능.
판매가와 할인판매가를 동시에 표시할 때 판매가에 취소선을 적용하고 싶어요
쇼핑몰 설정 > 상품 설정 > 상품 보기 설정 > 상품 정보 표시 설정에서 소비자가 취소선 표시 옵션 사용함 선택. 다만, 해당 기능 사용함에도 표시되지 않으면 디자인 점검 필요.
