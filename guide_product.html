일반등록

상품등록 > 일반등록 메뉴에서 항복별 설정 입력하고 상품 등록 버튼을 누르면 상품 등록 완료

주의 단계 기준 : 상품 8만개 ~ 10만개 미만 / 분류 2천개 ~ 3천개 미만

경고 단계 기준 : 상품 10만개 이상 / 분류 3천개 이상

엑셀 등록

메뉴 경로 : [관리자] 상품 > 상품 관리 > 상품 엑셀 관리
1. 상품등록

상품 엑셀 관리 메뉴에서 상품등록용 엑셀 다운로드 > 다운로드 한 엑셀 수정 후 엑셀 업로드

2. 상품수정

상품목록 메뉴에서 기본양식 선택하여 엑셀 파일 다운로드 > 엑셀 수정 후 상품 엑셀 관리 메뉴에서 엑셀 업로드

3. 옵션/재고수정

상품목록 메뉴에서 옵션/재고 초기화 양식 선택하여 엑셀 다운로드

> 옵션정보 수정 후 상품 엑셀관리 메뉴에서 옵션 수정 또는 재고 수정 탭에서 엑셀 업로드

상품재고 관리

메뉴 경로 : [관리자] 상품 > 상품관리 > 재고관리 > 상품재고 관리

상품재고 설정하기

상품등록/수정창 > 옵션/재고설정 항목에서 재고관리 사용 사용함 > 수량체크 기준 주문기준 > 재고수량 입력 > 품절표시 체크

재고관리 사용 : '사용함'

수량 체크 기준 : 운영 방식에 따라 '결제기준' 혹은 '주문기준' 으로 설정

※ 재고 수량체크 기준이 '결제기준'이면 실제 결제된 주문건으로 재고가 차감되므로 이를 원하지 않을 경우, 재고 수량체크 기준을 '주문기준'으로 변경 필요

품절표시 : 체크

※ '재고관리 사용'을 '사용함'으로 체크했더라도 '품절표시 사용'에 체크하지 않은 경우에는 재고가 0이 되어도 자동으로 품절처리가 되지 않아요.

분류관리

메뉴 경로 : [관리자] 상품 > 상품 관리 > 상품 분류 관리

쇼핑몰에 노출할 카테고리 설정

상품 분류 관리 메뉴에서 대분류 추가 버튼 클릭하여 대분류 추가하기

> 추가된 분류명 선택 후 우측 +버튼 클릭하여 중분류 생성하기 (같은 방법으로 소분류 생성 가능)

분류명 수정하기

생성된 대분류(또는 중분류) 선택 > 오른쪽 분류명 항목 수정

분류 삭제하기

삭제할 분류 선택 후 [ x 삭제 ] 버튼 클릭하여 삭제 가능

분류 이동하기

이동할 분류 선택 후 상/하로 드래그 앤 드롭 > 분류이동저장

분류별 진열

메뉴 경로 : [관리자] > 상품 > 상품 진열 > 분류별 진열

카테고리에 상품 진열하기

상품 > 상품목록 > 진열할 상품 선택 > 분류수정 > 팝업창에서 분류 선택 [+추가] > 선택한 상품 분류로 변경 > 확인

분류 진열 방법 수정하기

상품 > 분류관리 > 상품 분류 관리 > 수정할 분류 선택 > 상품진열 항목 사용자지정 또는 자동정렬+사용자지정 선택 후 하단 확인

분류 내 상품 순서 수정하기

분류별 진열 메뉴에서 분류 선택 후 검색 > 순서 변경할 상품 선택 > 선택한 상품의 진열순위를 [숫자 기재] 변경 > 확인

상품 재고 관리

메인분류 생성하기

상품 > 상품 진열 > 메인진열 메뉴에서 [메인분류 관리] 버튼 클릭 > 추가 또는 등록되어 있는 분류명 수정 가능

메인분류에 상품 진열하기

- 상품목록에서 진열방법 : 상품목록 메뉴에서 상품 선택 > 메인진열 수정 > 팝업에서 원하는 메인진열 진열함 선택 > 확인
- 메인진열에서 진열방법 : 메인진열 메뉴에서 원하는 메인분류 선택 후 검색 > 오른쪽 [상품추가] 버튼 클릭 > 원하는 상품 체크 후 선택

메인분류에서 상품순서 설정하기

메인진열 메뉴에서 원하는 상품 선택 > 하단에 선택한 상품의 진열순위를 [숫자 기재] 변경 > 확인

진열 예약

메뉴 경로 : [관리자] 상품 > 상품 진열 > 진열 예약

진열예약 설정하기

상품 > 상품진열 > 진열 예약 메뉴에서 오른쪽 예약등록 버튼 클릭 > 기간/상품 선택 후 진열할 카테고리 선택 > 저장

※ 참고사항

예약 기간 동안 설정한 상품 분류에 상품을 진열하는 기능

미진열 상품을 해당 기간에만 진열되도록 하고 싶은 경우 기간 설정 항목의 진열시작에서 상품 상태를 진열함으로 설하고

진열종료에서 상품 상태를 진열안함으로 설정 체크하여 설정 가능

한 예약건에서 설정할 수 있는 상품 분류는 1개, 메인 분류는 5개이며 동시에 다른 분류 예약 필요 시 예약 추가 등록 필요

메뉴 경로 : [관리자] 상품 > 상품관리 > 삭제 상품 목록

상품 삭제

상품목록에서 필요 없는 상품 선택 후 [x 삭제] 버튼 클릭하여 삭제

효과적인 상품관리와 원활한 쇼핑몰 운영을 위해 불필요한 상품은 완전 삭제 권장

삭제된 상품은 삭제일로부터 1년 후 완전 삭제됨

상품 복구

상품목록으로 다시 복구되며 삭제 상품 목록 리스트에서는 사라짐

완전 삭제

삭제 상품 목록에서 확인할 수 없으며, 완전 삭제 진행 시 복구 불가능

엑셀 다운로드

삭제 상품 목록을 엑셀 파일로 다운로드 가능, 암호화 후 압축된 파일로 제공됨

엑셀 다운로드 시 상품명의 HTML 태그를 삭제할지 여부를 옵션으로 설정 후 엑셀 파일 다운로드 요청
